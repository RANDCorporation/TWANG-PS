Having estimated the propensity scores, bal.table() produces a table that shows how well the resulting weights succeed in manipulating the control group so that its weighted pretreatment characteristics match, or balance, those of the unweighted treatment group if estimand = "ATT" or the control and treatment groups so that the weighted pretreatment characteristics match, or balance, with one another if estimand = "ATE". By default, the bal.table() function uses the value of estimand set with the ps() function call.
